# Добавление удалённого репозитория

<br/>

### Синтаксис команды
<p class="large">

```
git remote add <shortname> <url / path>
```

</p>

<br />

### Пример использования

<p class="small">

```
git remote add origin https://github.com/taiga-family/taiga-ui.git
```

</p>
